# Angular E-Commerce Application

A responsive e-commerce web application built with Angular that allows users to browse products, view product details, and manage a shopping cart.

## Technologies

- Angular
- TypeScript
- RxJS
- Reactive Forms
- Angular Router
- HttpClient
- HTML
- CSS

## Features

- Product listing
- Product details page
- Shopping cart functionality
- API integration using HttpClient
- Route protection using Angular Guards
- Reactive forms for user input
- Loading states and error handling
- Responsive design
